你可以使用本页面的代码覆盖manifest.json中的配置，查看浏览器几个页面被覆盖后的效果。

manifest.json中覆盖新标签页配置

"chrome_url_overrides" : {
    "newtab": "override.html"
}

manifest.json中覆盖历史记录页配置

"chrome_url_overrides" : {
    "history": "override.html"
}

manifest.json中覆盖下载页配置

"chrome_url_overrides" : {
    "bookmarks": "override.html"
}
